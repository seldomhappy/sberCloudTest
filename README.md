#Прототип консольного приложения для сокращения строк

Консольная инструкция:

```console
foo@bar:~$ make
foo@bar:~$ make help
```

Сборка и выполнение приложения:
```console
foo@bar:~$ make run
```

Вызов тестов:
```console
foo@bar:~$ make test 
```

Сборка бинарного файла:
```console
foo@bar:~$ make build 
```


>в текущей реализации поддерживаются только ASCII-символы
